Projects

Chatable

Chatable is the cumulative end project for a web-app coding bootcamp. We created an application to help students learn languages in a social, fun and non-traditional way. Chatable matches users with compatible language proficiencies so that they can engage in casual, fun and enriching conversation to practice their language skills. I worked with 2 other students (using Github for source control) to build this website from scratch. The project was created with Ruby on Rails, Html, CSS, Boostrap and a touch of AJAX. Main features include customizable user profiles, instant profile searching, recommended matches based on compatibility, and direct messaging between users. We deployed our app with heroku, and the entire code base is available on github.

Grocery List App

This is a personal project I have been building in my free time. This web application helps you shop smarter by customizing more intuitive ane efficient grocery lists. By recording your shopping patterns, it customizes your grocery lists for each of the stores you visit, and organizes the items within the list to match your path through the store (i.e. produce, then dairy, then frozen foods ect). The application is design and optimized to work both on a desktop and mobile browser, so that you can create your list at home, and then access it on your phone in the store. The project is built using Ruby on Rails, implementing a MySQL database. The frontend utilizes Bootstrap and AngularJS. Using the Facebook API, users can login with their Facebook accounts. The code base is available on github.
